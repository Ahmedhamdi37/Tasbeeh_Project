# Tasbeeh Project

A digital Islamic counter application for daily remembrance and worship practices.

## Overview
Tasbeeh is a spiritual application designed to help users keep track of their daily Islamic prayers, recitations, and supplications with an easy-to-use counter interface.

## Features
- Digital counter for Islamic recitations (Tasbeeh)
- Reset functionality
- Counter history tracking
- User-friendly interface
- Dark/Light theme support
- Responsive design

## Technologies Used
- JavaScript
- HTML5
- CSS3

## Getting Started
1. Clone the repository
2. Open `index.html` in your web browser
3. Start using the counter

## How to Use
1. Enter your intention or recitation
2. Use the counter buttons to track your recitations
3. Reset the counter when needed
4. Keep track of your daily progress

- Simple counter interface
